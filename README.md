# java raspberry pi clock
라즈베리파이 시계+체온계

연구실에 있던 재료들 주섬주섬 주워다 만든

출입자 체온 체크를 위한 장치

## 재료
 - Nextion NX4827T043_011 디스플레이
 - raspberry pi Zero W
 - fever click (MAX30205) 체온센서
 
## 특징
 - I2C및 UART R/W
 - 오차범위 ±5ms 이내의 정확한 시계
 - MAX30205센서는 표피 온도만 측정하여 30도 안밖의 값을 출력하므로 36.5도를 보고싶으면 다른 센서를 사용할것
